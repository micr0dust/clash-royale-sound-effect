# clash-royale-sound-effect

皇室戰爭大部分卡牌的出場音效，有些新卡的音檔資源還找不到所以沒聲音，之後再找找看。而攻擊、移動或其他音效之後會再更新，畢竟這是我今天下午太閒剛做的，所以還沒想這麼多。
